Oauth2 Demo

Application B (localhost:8080) has protected resource http://localhost:8080/api/accounts/default which can only be access by authorized users

Application A (localhost:8081) wants to access this resource

Redirect is made to application B, the user has to login in application B (user: demo, pass:123456) and confirm access to application A

After that redirect is made back to application A with temporary access code

Then application A requests token using separate HTTP connection and when the token is granted makes call to application B REST resource
